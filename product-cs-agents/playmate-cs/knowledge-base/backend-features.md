# 陪玩搭子组局系统 - 后台管理功能详解

## 系统概述

**系统名称**: 陪玩搭子组局系统  
**后台地址**: https://r18.qianchongxue.cn/  
**演示账号**: admin / admin888  
**H5端**: https://r18.qianchongxue.cn/h5

这是一个面向陪玩、搭子、组局场景的O2O服务平台，连接服务提供者（达人）与需求方（客户），支持多种社交娱乐场景的预约服务。

---

## 后台功能模块总览

系统后台包含 **22个一级菜单模块**，覆盖数据概况、达人管理、订单管理、营销管理、财务管理、系统配置等全链路运营功能。

---

## 一、数据概况模块

### 1.1 数据概况
- **功能定位**: 平台核心数据仪表盘
- **核心功能**:
  - 多维度数据筛选：今日、近7日、近30日、今年、自定义时间
  - 区域数据展示：已开通省份、城市、区县统计
  - 数据大屏入口：可视化数据展示，便于招商展示

### 1.2 达人统计
- **功能定位**: 达人数据分析与排行
- **核心功能**:
  - 达人数据列表：头像、姓名、信用分、评分、等级
  - 服务指标统计：服务时长、在线时长、业绩、加钟率
  - 订单指标统计：总订单量、已完成单量、退单率、总拒单数
  - 数据导出功能
  - 地图数据查看

## 四、达人管理模块

- **功能定位**: 平台服务提供者（达人）全生命周期管理
- **达人状态筛选**:

  - 归属信息：所属经纪人、所属代理商、挂靠门店

- **核心操作**:
  - 新增达人
  - 编辑达人信息
  - 删除达人
  - 取消授权
  - 修改代理商
  - 设置挂件

### 4.2 达人等级

- 等级权益设置

### 4.3 达人设置

## 六、分销体系模块

### 6.1 分销员管理
- **功能定位**: 平台分销推广人员管理
- **子模块**:

### 6.2 经纪人管理

- **子模块**:

  - **经纪人设置**: 经纪人规则配置
- **核心字段**: 用户ID、头像、昵称、姓名、手机号、备注信息、申请时间、状态
- **核心操作**: 新增经纪人、编辑、删除

### 6.3 业务员管理
- **功能定位**: 线下业务拓展人员
- **子模块**:
  - **业务员审核**: 业务员入驻申请审核
  - **业务员数据**: 业务员业绩统计
  - **业务员设置**: 业务员规则配置
- **核心字段**: 用户ID、头像、昵称、姓名、手机号、所属代理商、提成比例、申请时间、状态
- **核心操作**: 设置提成比例、删除提成比例

### 6.4 渠道商管理
- **功能定位**: 推广渠道合作伙伴
- **子模块**:
  - **渠道商审核**: 渠道商入驻申请审核
  - **渠道类目**: 渠道分类管理
  - **渠道财务**: 渠道商财务统计
  - **渠道商设置**: 渠道商规则配置
- **渠道码管理**:
  - 渠道码管理：生成/管理推广码
  - 渠道码统计：推广数据统计
- **核心字段**: 用户ID、头像、昵称、姓名、手机号、申请渠道、所属代理商、所属业务员、提成比例、可提现佣金、总佣金、备注信息、申请时间、状态

### 6.5 代理商管理
- **功能定位**: 区域代理合作伙伴
- **子模块**:

  - **代理商数据**: 代理商业绩统计

## 核心业务流程

### 1. 达人入驻流程
```
达人申请 → 资料审核 → 实名认证 → 授权开通 → 上架服务
```

### 2. 订单服务流程
```
客户下单 → 达人接单 → 服务进行 → 服务完成 → 评价结算
```

### 3. 分润结算流程
```
订单完成 → 平台抽成 → 达人分成 → 经纪人分润 → 代理商分润
```

## 关键术语解释

| 术语 | 说明 |
|------|------|
| 达人 | 平台服务提供者，提供陪玩、组局等服务 |
| 加钟 | 服务时间延长/加时 |
| 渠道商 | 推广渠道合作伙伴，通过渠道码推广获客 |
| 代理商 | 区域代理合作伙伴，管理区域内达人/渠道商 |
| 经纪人 | 达人管理者，负责达人招募与管理 |
| 业务员 | 线下业务拓展人员，隶属于代理商 |
| 分销员 | 推广分销人员，通过分享推广获客 |
| 渠道码 | 渠道商专属推广二维码/链接 |
| 物料 | 平台虚拟商品或实物商品 |
| 信用分 | 达人信用评分体系 |
| 加钟率 | 客户选择加时服务的比例 |
| 退单率 | 订单被取消的比例 |
| 拒单数 | 达人拒绝接单的数量 |
| 挂靠门店 | 达人所属的线下门店 |

## 客服场景覆盖

基于以上功能，客服Agent可处理以下场景：

### 1. 入驻咨询
- 达人入驻条件、审核进度、认证问题
- 经纪人/业务员/渠道商/代理商入驻流程
- 分销员申请门槛费说明

### 2. 订单问题
- 订单状态查询、退款申请、异常处理
- 加钟订单问题
- 拒单申诉处理
- 订单评价与投诉

### 3. 服务咨询
- 7大服务分类说明（K歌酒吧、一日游、剧本杀、电竞网咖、探店达人、逛街电影、露营搭子）
- 预约流程、费用标准
- 服务方式（上门/到店）

### 4. 分润结算
- 佣金计算规则（一级/二级提成）
- 提现问题、结算周期
- 渠道商/代理商分润查询
- 分销员佣金统计

### 5. 账号问题
- 登录问题、密码找回
- 信息修改、实名认证
- 权限问题、角色切换

### 6. 投诉建议
- 服务质量投诉
- 达人违规行为举报

### 7. 安全事件
- 求救通知处理
- 紧急联系人变更
- 线下服务安全保障

### 8. 分销推广
- 渠道码生成与使用
- 推广数据统计
- 分销员等级与权益

### 9. 门店相关
- 门店入驻申请
- 门店与达人挂靠关系
- 门店服务范围设置

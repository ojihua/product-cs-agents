# 🎭 Product CS Agents - 项目展示

> 三个专业级AI客服Agent，为不同行业产品提供智能客服解决方案

---

## 🌟 项目亮点

- ✅ **3个专业Agent** - 覆盖运势、服务、婚恋三大领域
- ✅ **完整知识库** - 超过1150行的产品知识
- ✅ **真实演示环境** - 4套在线演示系统
- ✅ **开箱即用** - 一键启动，快速部署
- ✅ **开源免费** - 基于OpenClaw框架

## 🚀 快速开始

### 方式1: 启动单个Agent

```bash
# 运势测算
cd agents/fortune-cs && ./start.sh

# 上门预约
cd agents/door-service-cs && ./start.sh

# 婚恋交友
cd agents/marriage-dating-cs && ./start.sh
```

### 方式2: 启动所有Agent

```bash
./agents/start-all.sh
```

### 方式3: 推送到GitHub

```bash
./push-to-github.sh YOUR_GITHUB_TOKEN
```

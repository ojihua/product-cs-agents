## 🆘 常见问题

### Q1: 系统支持哪些行业？
A: 系统适用于各类上门服务行业，包括但不限于：足疗理疗、美甲美睫、化妆造型、家政保洁、上门维修、上门洗车等。通过后台"服务业文字修改"功能，可自定义行业名称。

### Q2: 如何预约服务？
A: 用户端操作流程：
1. 选择服务类型（上门/到店）
2. 选择服务项目
3. 选择预约时间
4. 选择技师（可选）
5. 确认地址
6. 支付订单
7. 等待技师上门服务

### Q3: 技师如何入驻？
A: 技师入驻流程：
1. 下载技师端APP或关注公众号
2. 提交入驻申请（身份证、技能证书等）
3. 平台审核
4. 签署电子合同
5. 开始接单

### Q4: 如何保障技师安全？
A: 系统提供多重安全保障：
- 实时定位追踪
- 一键报警功能
- 服务过程录音
- 虚拟号码通话（保护隐私）
- 信用分评价体系

### Q5: 代理如何分成？
A: 代理分成机制：
- 平台可设置代理分成比例
- 代理可独立设置技师比例（需平台审核）
- 支持多级分销返佣
- 数据实时可查，支持即时提现

### Q6: 系统技术架构？
A: 
- 后端：ThinkPHP框架
- 前端：uni-app（适配小程序+H5+APP）
- 服务器：CentOS 7.9 + Nginx1.22.1 + MySQL5.6 + PHP7.2 + Redis
- 源码全开源，支持二次开发

### Q7: 如何成为代理/渠道商？
A: 
1. 联系平台提交申请
2. 审核通过后开通代理账号
3. 获得独立后台管理权限
4. 开始发展业务，获得分成

### Q8: 订单出现问题怎么办？
A: 订单问题处理：
- 订单转单：可将订单转给其他技师
- 订单退款：支持线上退款申请
- 异常单处理：平台介入处理
- 私单预警：系统自动监测异常订单

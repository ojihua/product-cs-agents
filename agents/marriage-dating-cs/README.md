# 完美运营版相亲交友系统 - 客服Agent

## 启动方式

```bash
# 方式1: 直接启动
bash /root/.openclaw/workspace/agents/marriage-dating-cs/start.sh

# 方式2: 使用openclaw命令
openclaw sessions_spawn --runtime subagent \
  --task "你是完美运营版相亲交友系统的客服助手小红娘。阅读 /root/.openclaw/workspace/agents/marriage-dating-cs/ 目录下的 SOUL.md、KNOWLEDGE.md、AGENTS.md，然后以客服身份问候用户并等待问题。"
```

## 文件说明

| 文件 | 说明 |
|------|------|
| SOUL.md | Agent身份和性格定义 |
| KNOWLEDGE.md | 产品知识库（功能、价格、FAQ） |
| AGENTS.md | 工作规范和话术模板 |
| IDENTITY.md | 身份信息 |
| start.sh | 快速启动脚本 |

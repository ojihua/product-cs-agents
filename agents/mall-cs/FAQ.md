# 联动2+1多功能商城系统 - 常见问题解答

## 🔐 登录问题

### Q1: 忘记密码怎么办？
**A**: 
1. 在登录页面点击"忘记密码"
2. 输入注册手机号
3. 获取短信验证码
4. 设置新密码
5. 使用新密码登录

### Q2: 验证码看不清怎么办？
**A**: 
- 点击验证码图片刷新
- 更换浏览器尝试
- 清除浏览器缓存
- 检查网络连接

### Q3: 账号被锁定怎么办？
**A**: 
- 等待30分钟后自动解锁
- 联系客服解锁
- 通过微信客服：dadifengqi

## 🛍️ 商品问题

### Q4: 如何发布商品？
**A**: 
1. 登录后台
2. 进入"商品管理" → "商品发布"
3. 填写商品信息（名称、分类、价格、库存）
4. 上传商品图片
5. 填写商品详情
6. 点击"保存并上架"

### Q5: 商品分类最多几级？
**A**: 
- 支持两级分类
- 一级分类：如"服装"、"食品"
- 二级分类：如"男装"、"女装"

### Q6: 商品图片有什么要求？
**A**: 
- 格式：JPG、PNG
- 大小：单张不超过2MB
- 尺寸：建议800x800像素
- 主图：至少1张，最多5张

### Q7: 如何设置商品规格？
**A**: 
1. 在商品发布页面
2. 点击"添加规格"
3. 设置规格名称（如"颜色"、"尺寸"）
4. 添加规格值（如"红色"、"XL"）
5. 设置各规格的价格和库存

## 📦 订单问题

### Q8: 如何查看订单？
**A**: 
- 后台："订单管理" → "订单列表"
- 可按状态筛选：待付款、待发货、待收货、已完成

### Q9: 如何发货？
**A**: 
1. 进入"订单管理" → "待发货"
2. 找到需要发货的订单
3. 点击"发货"按钮
4. 选择快递公司
5. 填写物流单号
6. 确认发货

### Q10: 支持哪些快递公司？
**A**: 
- 顺丰速运
- 中通快递
- 圆通速递
- 申通快递
- 韵达速递
- 百世快递
- 邮政EMS
- 其他（可自定义添加）

### Q11: 如何处理退款申请？
**A**: 
1. 进入"订单管理" → "售后管理"
2. 查看退款申请
3. 审核退款原因
4. 同意或拒绝退款
5. 如同意，系统自动退款

### Q12: 订单可以导出吗？
**A**: 
- 支持导出Excel
- 可按时间筛选
- 可按状态筛选
- 进入"订单列表" → "导出"

## 👥 会员问题

## 🎁 营销问题

### Q19: 优惠券有哪些类型？
**A**: 
- 满减券：满XX元减XX元
- 折扣券：打XX折
- 新人券：新用户专享
- 通用券：无门槛使用

### Q20: 如何设置分销返利？
**A**: 
1. 进入"营销管理" → "分销管理"
2. 开启分销功能
3. 设置分销等级（一级、二级）
4. 设置佣金比例
5. 设置提现规则
6. 保存

### Q21: 拼团活动怎么设置？
**A**: 
1. 进入"营销管理" → "拼团活动"
2. 点击"新建拼团"
3. 选择商品
4. 设置拼团价
5. 设置成团人数（2-10人）
6. 设置活动时间
7. 发布

### Q22: 秒杀活动有什么要求？
**A**: 
- 商品库存充足
- 设置秒杀价格（低于原价）
- 设置秒杀时间段
- 设置每人限购数量

## 💡 2+1联动模式问题

### Q35: 什么是2+1联动模式？
**A**: 
推荐2人成为会员，第3人自动滑落到团队下级，形成自动补位的团队结构。

### Q36: 如何获得收益？
**A**: 
- 直推奖励：推荐会员的奖励
- 团队分成：团队业绩的分成
- 滑落奖励：自动滑落成员的奖励

### Q37: 团队可以无限发展吗？
**A**: 
- 理论上可以无限层级
- 实际收益取决于团队活跃度
- 平台可设置层级限制

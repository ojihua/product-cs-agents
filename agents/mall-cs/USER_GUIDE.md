## 🖥️ 后台管理（商家/管理员）

### 商品管理

#### 发布商品
1. 进入"商品管理" → "商品发布"
2. 填写商品信息：
   - 商品名称
   - 商品分类
   - 商品图片（主图、详情图）
   - 商品规格（颜色、尺寸等）
   - 商品价格（原价、现价）
   - 商品库存
   - 商品详情描述
3. 点击"保存并上架"

#### 商品分类管理
1. 进入"商品管理" → "商品分类"
2. 添加一级分类
3. 在一级分类下添加二级分类
4. 设置分类排序

### 订单管理

#### 处理订单
1. 进入"订单管理" → "订单列表"
2. 查看待发货订单
3. 点击"发货"按钮
4. 选择快递公司
5. 填写物流单号
6. 确认发货

#### 售后处理
1. 进入"订单管理" → "售后管理"
2. 查看退款/退货申请
3. 审核售后申请
4. 处理退款或换货

### 会员管理

#### 查看会员

## 💡 2+1联动模式详解

### 什么是2+1联动模式？

2+1联动是一种创新的团队裂变营销模式：

**模式原理**:
- 用户推荐2人成为会员
- 第3人自动滑落到团队下级
- 形成自动补位的团队结构
- 享受团队业绩分成

**收益构成**:
1. **直推奖励**: 直接推荐会员的奖励
2. **间推奖励**: 下级推荐会员的奖励
3. **团队分成**: 整个团队业绩的分成
4. **滑落奖励**: 自动滑落成员的奖励

**优势**:
- 自动补位，团队自动填充
- 无限裂变，收益持续增长
- 公平分配，团队平衡发展
- 双重收益，推荐+团队
